Conócenos

¿Quiénes Somos?

LBTech es una joven empresa dedicada al desarrollo de software web y la gestión de datos. Somos de Montevideo, Uruguay. Nuestro nombre se dio una tarde de marzo del 2022, en donde pensamos que debería ser algo relacionado a nuestro enfoque de trabajo y a las innovaciones. Somos La Base Tecnológica, el nombre fue unánime por todos los integrantes de la empresa que hacen esto posible.

Nuestra empresa se divide en varios equipos dedicados a tareas específicas como: equipo de programadores, desarrollo web y testers. Además, contamos con un Project Manager que se dedica únicamente a las estrategias de los diversos proyectos que surgen, como también de coordinar, guiar y ayudar a los diferentes equipos en su área.

Misión

Desarrollar y brindar software que satisfaga las necesidades de nuestros clientes, convirtiéndonos así, en su primera y única opción.

Visión

Tener un impacto en el mundo creando software empoderando los negocios de nuestros clientes, liderando el ámbito tecnológico.

Nuestros servicios

Brindamos servicios adaptados a las necesidades de cada cliente, priorizando la calidad del producto que ofrecemos en cada proyecto elaborado.

Desarrollo de Software Web

Nos basamos en el lenguaje de programación PHP, y utilizamos el IDE Visual Studio Code para el desarrollo de Software y su buen funcionamiento. Además utilizamos GitHub para desarrollar al unísono entre todos los integrantes.

Gestión de datos

Trabajamos con el IDE Visual Studio Code para la creación, modificación y actualizaciones de bases de datos, basadas en las peticiones de nuestros clientes, para finalmente implementarla en el servidor MySQL.

Creación de páginas web

Utilizamos el IDE Visual Studio Code para la creación de páginas web basándonos en las necesidades y deseos de nuestros clientes. Ofreciéndoles un mejor posicionamiento para su producto u empresa.

Posicionamiento en redes sociales

El servicio del equipo de Marketing y del Community Manager consta de analizar el mercado y el público objetivo para la distribución de productos de nuestro cliente. Además, de ofrecerle un mejor posicionamiento en las diferentes redes sociales, brindándole un mayor alcance a futuros clientes interesados en sus productos y servicios.
